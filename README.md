# clever-neuwagen.de

Clever-Neuwagen – React/Vite Frontend + Express API (Demo/Pilot).

## Entwicklung

```bash
npm install
npm run dev
```

- App: http://localhost:5175  
- API: http://localhost:3001  

## Build

```bash
npm run build
npm run start:prod
```
